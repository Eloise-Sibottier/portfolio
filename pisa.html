Régression linéaire — Modèle final

Nous avons estimé plusieurs spécifications (formes linéaires, log-variables, interactions) et retenu le modèle présentant le meilleur compromis fit/interprétabilité pour expliquer ln(PISA).

R² ajusté ≈ 0.717
Modèle retenu — cible : ln(PISA)
Forme retenue — explication
ln(PISA) = β₀ + β₁ · dépense_éducation
           + β₂ · ln(PIB/hab)
           + β₃ · scolarisation_primaire
           + β₄ · sous-alimentation
           + β₅ · OCDE
           + β₆ · (scolarisation_maternelle × OCDE) + ε

Pourquoi ce modèle ? Il offre le meilleur ajustement tout en restant interprétable : la transformation logarithmique stabilise la variance et permet d’interpréter β₂ (ln(PIB/hab)) comme une élasticité approximative (≈ % de variation de PISA pour 1% de variation du PIB/hab).

Interprétation simple du log :
  • Si β₂ = 0.12, une hausse de 1% du PIB par habitant ≈ +0.12% du score PISA (approx.).
  • On travaille sur ln(PISA) pour réduire l’effet des outliers et rendre les élasticités directement lisibles.
Sélection des variables

Variables gardées (effets significatifs ou pertinents):

  • ln(PIB/hab) — effet positif robuste
  • Sous-alimentation — effet négatif significatif
  • Scolarisation maternelle (avec interaction OCDE)
  • Dépense éducation, scolarisation primaire (inclus pour contrôle)

Variables non significatives (éliminées) :

  • Indicateurs secondaires testés : certaines mesures régionales et indicateurs agrégés (ex. variable X, variable Y) — retirés pour parsimonie
  • Variables colinéaires détectées : retenu l’indicateur le plus robuste (ln(PIB/hab))
Interaction : scolarisation_maternelle × OCDE

L'interaction signifie que l'effet de la scolarisation en maternelle dépend du groupe OCDE/non-OCDE. Concrètement : la scolarisation maternelle a un effet plus marqué (bénéfice plus élevé) sur les scores PISA dans les pays hors OCDE ; dans les pays OCDE l’effet existe mais est atténué.

Conclusion

Le modèle retenu offre une synthèse robuste : la richesse et la nutrition sont des déterminants majeurs des scores PISA, tandis que les effets des politiques éducatives (dépense) nécessitent une interprétation prudente. La scolarisation maternelle paraît particulièrement bénéfique dans les pays hors OCDE.
